# Sistema de Diseño BIGA 2026
## Concepto: Pizzería Artesanal Premium (Calidez + Precisión)

1. Estética General:
   - Evitar diseños planos. Usar "Soft Shadows" (sombras difuminadas) para dar profundidad.
   - Bordes redondeados en cards (rounded-2xl o 3xl).
   - Fondos: Usar grises muy claros (bg-slate-50) en lugar de blanco puro para el fondo de página.

2. Paleta de Colores:
   - Primario (Botones/Acentos): Gradiente de Naranja (#FF8A00 a #FF4D00).
   - Éxito: Verde Albahaca (#059669).
   - Alerta: Rojo Salsa (#DC2626).
   - Sidebar: Negro Carbono (#1F1F1F).

3. Reglas de Layout (No-Scroll):
   - Reducir paddings verticales en headers.
   - Cards de navegación: Layout horizontal (Icono a la izquierda, título y descripción a la derecha).
   - Gap vertical reducido entre filas para que el contenido quepa en 100vh.
